**Candy Tower Panic**

- เป็นเกม Puzzle ที่จะมีลูกอมสีต่างๆ ตกลงมาในกระดานพื้นที่ 10x20 และผู้เล่นต้อง Clear ลูกอม โดยการคลิกที่ลูกอมจำนวน 3 ชิ้นขึ้นไป ที่มีสีเดียวกัน และอยู่ติดกัน เมื่อ Clear ลูกอม ก็จะได้คะแนน 
- ลูกอมจะมีทั้งหมด 5 สี คือ แดง น้ำเงิน เขียว เหลือง และม่วง
- เกมมี 2 โหมด ได้แก่ Sweet Target (โหมดจำกัดลูกอม) และ Sugar Rush (โหมดจำกัดเวลา) 
- Sweet Target คือ ผู้เล่นต้องทำคะแนนให้ถึงเป้าหมาย (Goal score) ก่อนที่จำนวนลูกอมที่มี (Max spawn) นั้นหมด หรือก่อนลูกอมต่อกันเป็น Tower จนถึงยอดสูงสุดของกระดาน
- Sugar Rush คือ ผู้เล่นต้องทำคะแนนให้ถึงเป้าหมาย (Goal score) ภายในเวลาที่กำหนด (Time limit) หรือก่อนลูกอมต่อกันเป็น Tower จนถึงยอดสูงสุดของกระดาน
- เกมมี 3 ระดับความยาก ได้แก่ Easy, Medium และ Hard ซึ่ง Goal score, Time limit และ Max Spawn ก็จะเปลี่ยนไปตามระดับความยากที่เลือก

**Engine ที่ใช้**
- Python 3 และ Pygame

**การ Run**

1. ติดตั้ง Python 3.10 ขึ้นไป
2. ติดตั้ง Pygame: pip install pygame
3. python candy_tower_panic.py

**ปุ่มควบคุม ภายในเกม (Controls)**
- เมาส์คลิกซ้าย - Clear ลูกอม
- R - เริ่มเล่นใหม่ (Retry)
- M - กลับสู่หน้าหลัก (Main menu)
- P - พักเกมชั่วคราว (Pause)
- ESC - ออกเกม
